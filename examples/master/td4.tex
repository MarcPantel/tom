\maketitle

Ce quatrième TD a pour but d'approfondir les techniques de programmation en Tom.
Nous abordons dans ce TD l'utilisation de symboles associatifs et de stratégies.

L'objectif est d'implanter un pico-outil d'analyse statique.

\section{Expressions}
Proposer une grammaire permettant de représenter des expressions composées de:
\texttt{Var, Cst, Let, Seq, If, Print} et \texttt{Plus}.

\section{Programme}
Écrire les programmes
\begin{itemize}
\item \texttt{a := 1}
\item \texttt{b := a+2 ; print b}
\item \texttt{i := 0 ; if i!=10 then Print(i) ; i := i+1}
\end{itemize}

\section{Optim 1}
Proposer un système de réécriture, décrit en Tom, permettant d'optimiser les \texttt{Seq} et les \texttt{If}.


\section{Collecter des variables}
\begin{itemize}
\item écrire une stratégie qui affiche \texttt{cst} lorsqu'un noeud \texttt{Cst} est rencontré
\item écrire une stratégie qui affiche toutes les constantes contenues dans un terme
\item écrire une stratégie qui fait l'identité  lorsque \texttt{Cst} est rencontré
\item écrire une stratégie qui affiche une noeud
\item en utilisant les 2 stratégies précédentes, écrire une stratégie qui
affiche toutes les constantes contenues dans un terme
\end{itemize}

\section{Renommage de variables}
\begin{itemize}
\item écrire une stratégie qui renomme en \texttt{\_name} le nom d'une variable
\item renommer toutes les variables d'une arbre
\item renommer, en affichant les étapes intermédiaires, toutes les variables d'un arbre
\end{itemize}

\section{Optim 2}
\begin{itemize}
\item refaire l'optimiseur de \texttt{If} en utilisant une stratégie
\end{itemize}
